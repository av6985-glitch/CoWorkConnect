* Dea Tahiraj

## Project Overview

CoWorkConnect is a web-based system that allows users to register, log in, and book available workspaces.
Administrators can manage workspace availability and monitor bookings.

## Design Patterns Implemented

* MVC (Model View Controller)
* Singleton Pattern
* Observer Pattern

## Technologies Used

* HTML
* CSS
* JavaScript (LocalStorage simulation)

## How to Run

1. Download or clone the repository.
2. Open `index.html` in any browser.
3. Register and login to test features.
